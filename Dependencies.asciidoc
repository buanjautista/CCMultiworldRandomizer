== Manual installation

Install https://wiki.c2dl.info/CCLoader[CCLoader 2], then download the *most recent*
https://github.com/CodeTriangle/CCMultiworldRandomizer/releases/latest[CCMultiworldRandomizer release] and place it
into the `assets/mods` directory. Then, download a compliant version (probably the most recent) of all of the mods
listed above. **Make sure you get the `.ccmod` file and not the source code zip for all mods downloaded.**

Continue with the setup process as outlined in the link:Setup[setup guide].
